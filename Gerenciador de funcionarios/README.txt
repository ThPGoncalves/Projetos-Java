Versão PT-BR
É necessário ter os pacotes do JavaFX novos instalados no computador.
Ao compilar o código e ao chamar o código para mim foi necessário dar o caminho dos módulos nos comandos abaixo, substitua pelo seu caminho de instalação
javac --module-path "C:\Users\user\Downloads\openjfx-21.0.2_windows-x64_bin-sdk\javafx-sdk-21.0.2\lib" --add-modules javafx.controls,javafx.fxml Funcionario.java
java --module-path "C:\Users\user\Downloads\openjfx-21.0.2_windows-x64_bin-sdk\javafx-sdk-21.0.2\lib" --add-modules javafx.controls,javafx.fxml Funcionario
